JavaScript对象与JSON的区别

Sep 16, 2016点击

JSON对象

  • JSON就是JavaScript Object Notation ，是js对象表示法，是js对象的严格子集
  • JSON要求所有的属性名必须加双引号，而对象其实不要求这个

为什么JSON规定要加上双引号呢？这是因为JSON是一个数据交换格式，他可以负责PHP的通信。PHP可以从数据库中拿到JSON，组合数据，然后交给前端。说白了，他就是一个数据交换媒介。所以是其他语言要求JSON要有双引号，他天生就是为了通信而生的

对象和JSON的区别就是引号.除了引号没区别
